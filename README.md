# Projeto Simplex

Este repositório contém os arquivos iniciais de um trabalho acadêmico sobre o método Simplex e otimização linear.

## Objetivo

O projeto pretende implementar uma solução para problemas de programação linear usando o método Simplex e apresentar uma análise do funcionamento básico do algoritmo.

## Estrutura inicial

- Documentação do enunciado do trabalho
- Anotações das aulas relacionadas a otimização e Simplex
- Código e testes a serem desenvolvidos em etapas futuras

## Próximos passos

1. Analisar o enunciado para definir requisitos e casos de uso.
2. Estudar as notas de aula para compreender a teoria do método Simplex.
3. Iniciar a implementação de uma versão básica do algoritmo.
4. Validar com exemplos simples de programação linear.

## Observação

Este README é um esboço inicial. O desenvolvimento do projeto ainda está em fase conceitual e será atualizado conforme o trabalho avance.
